# 💼 Payroll Management System

A Python console application that automates employee payroll calculations, including salary, bonus, tax deduction, and net salary.

---

## 📖 Overview

This project simulates a simple Human Resources (HR) payroll system. It allows users to enter employee information and automatically calculates bonuses, taxes, gross salary, and net salary.

The project focuses on writing clean, modular, and maintainable Python code.

---

## ✨ Features

- 👤 Employee Information Management
- 💵 Salary Calculation
- 🎁 Bonus Calculation
- 🧾 Tax Deduction
- 💰 Net Salary Calculation
- ✅ Input Validation
- ⚠️ Error Handling
- 📋 Employee Payroll Summary

---

## ▶️ How to Run

1. Clone the repository.

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Run:

```bash
python main.py
```
