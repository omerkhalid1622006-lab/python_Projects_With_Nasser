# 🏦 Bank Account Management System

A Python console-based banking system that simulates basic banking operations such as account registration, user authentication, deposits, withdrawals, and balance inquiries.

---

## 📖 Overview

This project was developed to practice Python fundamentals by building a simple banking application. It demonstrates the use of functions, dictionaries, file handling, loops, conditionals, and input validation.

---

## ✨ Features

- 👤 User Registration
- 🔐 Secure Login
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Account Balance
- 🚪 Logout
- ❌ Exit the Application
- ⚠️ Input Validation

---

## ▶️ How to Run

1. Clone the repository.

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Run:

```bash
python main.py
```
